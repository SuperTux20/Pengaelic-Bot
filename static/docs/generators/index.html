Some silly random generators! The fortune command takes output from Unix Fortune, I am not responsible for any offensive output it may generate. The "IMG" command is an Infinite Monkey Generator, and it'll automatically stop after 10 minutes.

p!name [number of names to generate] [upper syllable limit threshold] [lower syllable limit threshold]

p!ename [number of names to generate] [upper syllable limit threshold] [lower syllable limit threshold]

p!floridaman [alternative other state or region]

p!img <keyword to search for> [specfic alphabet/charset to use]
